# node-ts-template
A basic NodeJS template for NodeJS, Typescript, ESM &amp; Top-Level await built in. 

## Running Scripts:
- `npx tsx <file>`
- `node --experimental-strip-types <file>`

## Recommendations:
- Use modern node: `v22.6.0` or greater
- Make sure `tsx` is installed globally: `npm i tsx -g`
